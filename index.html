    Брендбук

    1. Как сейчас
    Брендбук – это один из основополагающих документов компании, регламентирующий миссию, цели и задачи бренда на рынке. Это своеобразная «Библия» бренда, на основании которой строится вся маркетинговая стратегия. Без данного документа реализация любой стратегии невозможна из-за отсутствия идейной основы, объединяющей различные инструменты и каналы коммуникации. При попытке реализации маркетинговой стратегии без брендбука, каждый канал и инструмент будет ориентироваться только на лидогенерационные метрики и не будет доносить до целевой аудитории суть бренда, его ценности, миссию, философию и индивидуальность. Таким образом, полноценного бренда без брендбука просто не может существовать. Исходя из предоставленной информации, в настоящий момент у компании отсутствует полноценный брендбук. Сформирован только логобук, в котором представлено визуальное оформление торговой марки. При этом, сама торговая марка, из-за отсутствия брендбука, не имеет содержания.

    2. Как будет
    Брендбук систематизирует идеологию бренда, что позволит использовать все каналы коммуникации для донесения этой идеологии для конечного потребителя и создания у потребителя устойчивой эмоциональной связи со всеми элементами бренда, в том числе и с фирменным стилем.

    3. Каким будет результат
    Брендбук, полностью отвечающий требованиям топ-менеджмента к будущей идеологии и позиционированию.

    SMM

    1. Как сейчас
    В настоящий момент социальные сети ведутся по остаточному принципу, что выражается в практически полном отсутствии полезного контента. Исходя из постов, опубликованных в последнее время, для социальных сетей не разрабатывался рубрикатор и контент-план. Их отсутствие сказывается на качестве ведения и отсутствии роста базы подписчиков. Продолжение работы в текущем режиме никогда не приведет к какому-либо результату.

    2. Как будет
    Будет разработан список рубрик, каждая из которых будет решать определенную задачу в рамках коммуникации с сегментом целевой аудитории. Исходя из рубрикатора, каждые две недели будет формироваться контент-план, содержащий конкретные посты, запланированные к публикации. В ходе работы, статистика по вовлеченности постов будет анализироваться. На основании проведенного анализа, рубрикатор будет корректироваться для повышения его эффективности.

    3. Как посчитать результат
    Для подсчета результативности будут использоваться следующие метрики:
    • Количество подписчиков;
    • Количество лайков, репостов и комментариев;
    • Количество заявок из социальных сетей;
    • Количество новых клиентов, приведенных из социальных сетей;

    4. Каким будет результат
    Живые страницы в социальных сетях, которые будут являться не только источником новых клиентов, но и способом коммуникации с целевой аудиторией. Более того, при корректном управлении и должном уровне доверия, данный инструмент может быть рычагом влияния на мнение аудитории в том числе и с помощью манипулирования комментариями.

    SERM

    1. Как сейчас
    SERM – управление репутацией компании. В настоящий момент работа с репутацией, в сравнении с остальными направлениями – одно из самых развитых. Тем не менее, существует ряд недоработок, способных повлиять на итоговый результат. В частности:
    • Идентичные отличные оценки;

    • Отсутствие проработки негатива

    • Отсутствие компании на популярных сайтах для отзывов
    irecommend.ru.
    otzovik.com.
    zoon.ru.
    apoi.ru.
    pravogolosa.net.

    2. Как будет
    Будет разработан план публикации отзывов для регулирования средней оценки предприятия на уровне 4,7-4,8 баллов на всех доступных площадках. Таким образом, оценка будет находиться на высоком уровне, при этом не вызывая подозрений относительно ее объективности. Все негативные отзывы будут публично обрабатываться от имени компании и, при необходимости, передаваться для доработки и сглаживания негатива к непосредственным представителям компании.

    3. Как посчитать результат
    • Количеством опубликованных отзывов;
    • Количеством обработанных негативных отзывов;
    • Количеством разрешенных конфликтных ситуаций;

    4. Каким будет результат
    Репутация компании будет поддерживаться на стабильно высоком уровне и подтверждаться действиями компании для урегулирования неоднозначных ситуаций. При использовании брендовых запросов в сети, пользователь будет сталкиваться с позитивными отзывами о компании на различных специализированных площадках, что будет являться дополнительным аргументом для обращения в компанию.

    Email

    1. Как сейчас
    Исходя из данных Roistat, за последние полгода была отправлена только две рассылки: о COVID и Поздравление с Новым годом 2020. Таким образом, можно сказать, что данный инструмент не используется.

    2. Как будет
    База клиентов будет сегментирована по доступным на первом этапе критериям. Для каждого сегмента будут выстроены email-воронки, содержащие в себе информационные, полезные и продающие письма.

    3. Как посчитать результат
    Для подсчета результативности будут использоваться следующие метрики:
    • Количество писем;
    • Процент открываемости писем и переходов на сайт;
    • Количество заявок;
    • Количество продаж;

    4. Каким будет результат
    Email будет превращен в полноценный канал лидогенерации и допродаж.

    SEO

    1. Как сейчас
    Исходя из данных Roistat, в настоящий момент SEO-продвижение исключительно неэффективно. Как показали данные Roistat, из более чем 2000 обращений, поступивших из SEO фактически новыми клиентами является только 16 компаний. Основная выручка в течение полугода формируется за счет допродаж по клиентам, которые стали постоянными.

    2. Как будет
    Семантическое ядро будет пересмотрено и очищено от нецелевых запросов, в результате чего количество обращений сократится, однако их качество возрастет. Это снизит неэффективную нагрузку на отдел продаж и повысит общую выручку компании.

    3. Как посчитать результат
    • Объем трафика;
    • Позиции в выдаче по целевым ключевым фразам;
    • Количество качественных обращений;
    • Количество продаж;

    4. Каким будет результат
    Большее количество новых клиентов, приведенных из канала. Большее количество денег. Меньшее количество некачественных обращений.

    PR

    1. Как сейчас
    В настоящий момент компания не занимается PR-направлением. В интернете не было найдено ни одной публикации в СМИ о компании, работа с лидерами мнений также не ведется.

    2. Как будет

    • Донесение до целевой аудитории ценностей бренда (позиционирование).

    Таргетированная реклама станет дополнительным источником заявок, а также способом укрепить транслируемую эмоциональную связь торговой марки с целевой аудиторией.

    Контекстная реклама

    1. Как сейчас
    В настоящий момент из-за некачественной настройки, данный рекламный канал работает с отрицательной доходностью и не приносит необходимого количества новых клиентов.

    2. Как будет
    Будет проработано семантическое ядро, максимально охватывающая целевые поисковые запросы и не затрагивающее околоцелевые, неэффективно расходующие бюджет. Предположительная архитектура рекламных кампаний будет основана на сегментации семантики по услугам компании:
    • Таможенный представитель
    • Транспортное экспедирование грузов
    • Терминальная обработка
    • Таможенное декларирование товаров и грузов
    • Таможенные услуги
    • Разрешительная документация
    • Транспортно-экспедиционное обслуживание
    При этом, для каждого сегмента будет использоваться точная посадочная страница, конкретно отвечающая на запрос пользователя
    Данная структура является предварительной и может быть усложнена в случае необходимости более точной сегментации поисковых запросов

    3. Как посчитать результат
    • Количество обращений
    • Количество новых клиентов
    • Количество продаж
    • Количество денег ROI

    4. Каким будет результат
    Прибыльные рекламные кампании со стабильной эффективностью, которая постоянно растет.
